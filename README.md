# edu-flow
EduFlow é uma plataforma desenvolvida para o processo seletivo da Agenda Edu, criada para otimizar o gerenciamento acadêmico. O sistema permite que professores cadastrem e acompanhem tarefas, enquanto alunos acompanham seus prazos, status e feedback de forma prática e organizada, promovendo uma experiência educacional mais eficiente e colaborativa.
